# NovaWatch Landing Page

This project is a responsive product landing page developed as part of my Web Development Internship at Codveda Technologies.

## 🚀 Overview
The website showcases a smartwatch product called NovaWatch. It is designed with a modern and clean layout to provide a good user experience.

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📱 Features
- Fully responsive design
- Modern user interface
- Navigation bar and hero section
- Features section with cards
- Product highlights section
- Call-to-action section
- Footer with contact details

## 🎯 Purpose
This project was created to demonstrate my front-end development skills and ability to build responsive websites using HTML and CSS.
