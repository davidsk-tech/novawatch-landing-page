# NovaWatch — Landing Page

This project is a responsive product landing page developed as part of my Web Development Internship at Codveda Technologies.

A modern, responsive product landing page for **NovaWatch Series 1**, built with pure HTML and CSS (no JavaScript).

---

## Files

index.html   — Page structure and content  
style.css    — All styling, layout, and animations  
README.md    — Project documentation  

---

## Usage

No build step or dependencies required. Just open `index.html` in any modern browser.

Option 1 — double-click index.html in your file explorer  

Option 2 — serve locally with Python  
python -m http.server 8000  
then visit http://localhost:8000  

Note: Google Fonts are loaded via CDN. An internet connection is needed for correct typography. The page is fully functional offline — fonts will fall back to system sans-serif.

## Sections

1. Nav
